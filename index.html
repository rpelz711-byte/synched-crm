Cloudflare Worker setup

The worker.js file in this project needs these environment variables set in Cloudflare:
SUPABASE_URL   SUPABASE_KEY   GEMINI_API_KEY   YOUTUBE_API_KEY

Go to: Cloudflare Dashboard → Workers & Pages → your worker → Settings → Variables and Secrets → Add variable

Then add a Cron Trigger: Settings → Triggers → Cron Triggers → 0 7 * * * (runs 7am UTC daily)
